# Information Security Guidelines

## Passwords
Never reuse passwords across systems. Store passwords securely in the approved
password manager, never in plain text files or spreadsheets. Passwords must be at
least twelve characters long. This is a fictional sample document used only to
exercise the lab.

## Multi-factor authentication
Enable multi-factor authentication (MFA) on every account that supports it. MFA
significantly reduces the risk of account compromise if a password is stolen.

## Phishing
Be cautious with unexpected emails asking you to click links or share credentials.
Report suspected phishing messages to the security team immediately.

## Data classification
Classify documents as Public, Internal, or Confidential and handle each according
to its classification. Confidential data must not be shared outside the organisation.
